# Order System

## Start

```bash
npm install
pm2 start ecosystem.config.cjs
```

## Routes

- Admin login: `/admin`
- Admin shop A: `/admin/shop-a`
- Admin shop B: `/admin/shop-b`
- Worker shop A: `/worker/shop-a`
- Worker shop B: `/worker/shop-b`

Shop upstream accounts and cookies are in `shops.config.json`.
